Modal - Custom Buttons

The Hello World demo shows the most basic use of BootstrapPlus.Modal. It has a basic header, body, and Close button. Now let's see how to make custom buttons.

The code is fairly simple. BootstrapPlus.Modal expects a key named buttons in the configuration object that contains an object whose keys will be the names of the buttons to place on the dialog. Each button will have a handler method, and an optional type. type is where you can tell BootstrapPlus.Modal if the button is a primary button or not. Here's a sample:

var modal = new BootstrapPlus.Modal({
  header: "Message For You",
  body: "<p>Below are custom buttons. They do stuff!</p>",
  buttons: {
    "What's My ID?": { handler: function() { alert(modal.getId()); }},
    "Say Hi": { handler: function() { alert("Hello."); }},
    "Close": { type: "primary", handler: function() { modal.close(); }}
  }
});
