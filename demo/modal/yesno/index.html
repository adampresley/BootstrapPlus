Modal - YesNo

The YesNo demo shows how to use a specialized version of the modal dialog that shows a header, body, Yes and No buttons. With this version you can also provide a single handler function for when either button is clicked and the button clicked will be passed to it in the form of "yes" or "no".

new BootstrapPlus.Modal.YesNo({
  header: "Confirmation",
  body: "<p>Are you sure?</p>",
  handler: function(button) {
    if (button === "yes") {
      alert("YES!");
    } else {
      alert("NO!");
    }
  }
});
